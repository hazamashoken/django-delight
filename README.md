## Getting Started
### Requirements
* python 3.10.9
* pipenv

Run the following commands to start the server:
```
pipenv install django django-ninja; pipenv shell; cd djangodelights; python3 manage.py runserver;
```
or
```
sh autoinstall.sh
```
